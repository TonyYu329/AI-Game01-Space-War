# 星河战机 - Space War Game

> 2026-04-24 VSCode+Cline+Deepseek 首次自主创作，后续由claudecode+DeepSeek修改发布。

星河战机是一款基于 Canvas 的太空射击游戏，包含多个角色主题的游戏入口文件。游戏支持鼠标/键盘/触屏操作，具有丰富的视觉特效和音效系统。

## 游戏入口

index.html

## 操作方式

- **鼠标移动**：控制飞船位置
- **鼠标左键 / J 键**：发射子弹
- **鼠标右键 / K 键**：发射导弹
- **方向键 / WASD**：移动飞船
- **触屏**：滑动控制飞船，点击发射子弹

## 技术栈

- HTML5 Canvas
- 原生 JavaScript（无框架依赖）
- Web Audio API（音效生成）
- 响应式设计（适配 PC 和移动端）
