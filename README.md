# 星河战机 - Space War Game

> 2026-04-24 VSCode+Cline+Deepseek 自主创作

星河战机是一款基于 Canvas 的太空射击游戏，包含多个角色主题的游戏入口文件。游戏支持鼠标/键盘/触屏操作，具有丰富的视觉特效和音效系统。

## 游戏入口

| 文件 | 角色 | 特色词语 |
|------|------|----------|
| `xueer.html` | 雪儿 | 小雪、雪儿、老师、幼师、宝贝…… |
| `xiaokeai.html` | 小可爱 | 小可爱、小傻瓜、秋瑾、裱花师…… |
| `nana.html` | 娜娜 | 娜娜、小马、建娜、娜娜医生…… |

## 操作方式

- **鼠标移动**：控制飞船位置
- **鼠标左键 / J 键**：发射子弹
- **鼠标右键 / K 键**：发射导弹
- **方向键 / WASD**：移动飞船
- **触屏**：滑动控制飞船，点击发射子弹
